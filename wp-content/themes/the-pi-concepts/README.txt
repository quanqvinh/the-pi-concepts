== Description ==

The Pi Concepts is a custom-built, elegant WordPress block theme created from scratch for the Pi Concepts project. It features a modern design, custom color options, and is fully compatible with the latest WordPress block editor.

Key Features:
* Block-based theme structure
* Custom color palette and elegant design
* Full Site Editing (FSE) support
* Responsive and accessible layout
* Clean, semantic code

== Installation ==

1. Upload the theme files to `/wp-content/themes/the-pi-concepts/`
2. Activate the theme via the WordPress admin under Appearance > Themes
3. Customize your site using the Site Editor and available block options
